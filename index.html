🔐 ¿Qué es HWallet?

HWallet es una wallet virtual en $HCASH y TON que permite:

  • Transferencias entre usuarios sin comisiones internas
  • Creación de tareas de promoción
  • Ganancias reales al participar en tareas
  • Autogestión total sin admins ni terceros

⚙️ Enviar dinero

Usa el comando hh para transferencias rápidas:

  • hh 2 hcash @usuario — envía 2 HCASH
  • Responde a un mensaje y escribe hh 3 hcash
  • Funciona también con TON (hh 1 ton)

🛠️ Crear tareas de promoción

  1. Presiona el botón Servicios y elige Promover Grupo
  2. Escribe el nombre de tu tarea
  3. Envía el link del grupo o canal
  4. Define la recompensa por usuario
  5. Indica la cantidad deseada de usuarios
  6. Confirma y publica tu tarea

Solo pagarás si los usuarios cumplen la acción.

📲 Completar tareas

  1. Ve a Servicios → Tareas disponibles
  2. Elige una y haz clic en el link
  3. Una vez te unas, presiona ✅ Ya me uní
  4. El bot validará automáticamente y te pagará

🧾 Tu perfil

  • Saldo en HCASH y TON
  • Respaldo bloqueado en tareas
  • Tareas completadas
  • Tareas activas como creador
  • Tu ID de usuario

💵 Retiros y depósitos

  • Usa el botón Depositar y sigue instrucciones
  • Recuerda incluir tu memo al depositar
  • Para retirar, el bot te pedirá la dirección
  • Requiere saldo en TON para fee (+0.1 extra)

📋 Comandos disponibles

  • hh 3 hcash @usuario → Envía HCASH
  • hh 2 ton → Envía TON (al responder mensaje)
  • mywallet → Mostrar perfil
  • hh ref → Obtener tu link de referido
  • start o /start → Ver menú

🎁 Sistema de referidos

  • Usa hh ref para obtener tu link
  • Ganas 0.25 HCASH si tu referido publica una tarea
  • Ganas 0.02 TON cada vez que retira
  • Las ganancias se suman a tu balance

🆘 Soporte directo

  • Desde el menú, presiona Soporte
  • Envía tu mensaje directamente al equipo
  • Recibirás una respuesta personalizada desde el bot
  • El historial se guarda para futuras referencias
